# Prácticas de Programación - Corte 1

Descripción: Colección de ejercicios introductorios al lenguaje C. Se centra en el dominio de la sintaxis básica, operadores aritméticos, lógicos y el comportamiento de los operadores a nivel de bits.

Conceptos Técnicos Aplicados:
> - Estructura básica de un programa en C (`stdio.h`, `main`).
> - Operadores aritméticos y de asignación compuesta.
> - Diferencia entre incremento/decremento prefijo y posfijo (`++a` vs `a++`).
> - Operadores lógicos y de comparación.
> - Operadores a nivel de bits (AND, OR, XOR, NOT, desplazamientos).
> - Formateo de salida con `printf`.

Stack: C, GCC.

Cómo ejecutar:
```bash
# Ejemplo para uno de los archivos
gcc prueba7.c -o prueba7
./prueba7
```

Notas de aprendizaje: Se practicó la base fundamental del lenguaje, entendiendo cómo los datos son manipulados a nivel binario y cómo el orden de los operadores afecta el resultado de las expresiones.
