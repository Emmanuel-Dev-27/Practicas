# Estructuras de Datos - Corte 3

Descripción: Implementación y estudio de estructuras de datos lineales utilizando punteros y gestión dinámica de memoria. Incluye implementaciones robustas para listas, pilas y colas.

Conceptos Técnicos Aplicados:
> - Nodo asimétrico/simétrico y punteros auto-referenciados.
> - Listas Enlazadas Simples (inserción, búsqueda, eliminación).
> - Listas Circulares.
> - Estructuras LIFO (Pilas) y FIFO (Colas).
> - Recursividad para liberación de memoria y manipulación de estructuras.
> - Uso de archivos de cabecera (.h) para la abstracción de tipos de datos (ADT).

Stack: C, GCC, Make (Makefile incluido).

Cómo ejecutar:
```bash
# Utilizando el Makefile incluido
make
./bin/programa_principal
```

Notas de aprendizaje: Se reforzó la capacidad de abstracción al separar la lógica de las estructuras de datos del programa principal, permitiendo la creación de librerías reutilizables.
