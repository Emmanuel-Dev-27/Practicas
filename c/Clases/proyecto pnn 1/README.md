# Proyecto PNN 1 - Aplicaciones Prácticas

Descripción: Colección de aplicaciones finales que integran diversos conceptos de programación en C, incluyendo lógica de juegos (Casino) y cálculos físicos (Distancia de Relámpago).

Conceptos Técnicos Aplicados:
> - Lógica condicional compleja y bucles de juego.
> - Generación de números aleatorios (`srand`, `rand`).
> - Entrada y salida formateada para interactuar con el usuario.
> - Cálculos físicos y conversión de unidades.
> - Validación de flujo de programa.

Stack: C, GCC.

Cómo ejecutar:
```bash
# Para el juego del Casino
gcc Casino.c -o Casino
./Casino

# Para el cálculo de distancia
gcc Distancia_relampago.c -o Distancia_relampago
./Distancia_relampago
```

Notas de aprendizaje: Estas aplicaciones sirvieron para consolidar el conocimiento en un contexto de resolución de problemas del "mundo real" y desarrollo de lógica de entretenimiento.
