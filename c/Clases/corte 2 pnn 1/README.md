# Prácticas de Programación - Corte 2

Descripción: Ejercicios avanzados centrados en la gestión de memoria y el uso de punteros en C. Cubre desde el acceso directo a direcciones de memoria hasta la gestión dinámica de recursos.

Conceptos Técnicos Aplicados:
> - Declaración y desreferenciación de punteros (`*` y `&`).
> - Relación entre arreglos y punteros (notación de arreglo vs notación de puntero).
> - Arreglos de punteros (ej. cadenas de texto).
> - Gestión dinámica de memoria (`malloc`, `free`).
> - Prevención de fugas de memoria (Memory Leaks).

Stack: C, GCC.

Cómo ejecutar:
```bash
# Ejemplo para un archivo de punteros
gcc punteros.c -o punteros
./punteros

# Ejemplo para un archivo de memoria dinámica
gcc memoria.c -o memoria
./memoria
```

Notas de aprendizaje: Se profundizó en el concepto de que "todo en C es una dirección de memoria", fortaleciendo la habilidad de manejar datos de forma eficiente y entender el stack vs heap.
