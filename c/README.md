# Prácticas de Lenguaje C - Desarrollo de Bajo Nivel

Este directorio contiene ejercicios enfocados en la gestión de memoria, algoritmos de álgebra lineal y criptografía básica utilizando el lenguaje C.

## Proyectos Destacados
- **[Calculadora Matricial](./CalculadoraMatricial)**: Operaciones avanzadas con matrices y memoria dinámica.
- **[Codificador XOR](./CodificadorXOR)**: Implementación de cifrado simétrico nivel bits.
- **[Clases y Laboratorios](./Clases)**: Ejercicios modulares sobre punteros, estructuras de datos y lógica fundamental.

Conceptos Clave: Punteros, Aritmética de direcciones, Gestión de Memoria (malloc/free), Estructuras de datos dinámicas.
