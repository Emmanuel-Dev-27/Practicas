# Prácticas de Python - Ecosistema de APIs y Aplicaciones

Descripción: Repositorio central que contiene diversas implementaciones de servicios web y aplicaciones de consola. El enfoque principal es el dominio de frameworks como Flask y FastAPI, junto con la gestión de diferentes motores de bases de datos.

Contenido Principal:
- **APIs con Flask**: Implementaciones con MySQL (Directo y SQLAlchemy) y SQLite.
- **APIs con FastAPI**: Sistema de usuarios con MongoDB y autenticación JWT.
- **Aplicaciones Web**: Gestores de tareas y listas de compras.
- **Fundamentos**: Ejercicios de lógica y tipado.

Stack: Python, Flask, FastAPI, MySQL, SQLite, MongoDB.

Notas de aprendizaje: Este conjunto de prácticas demuestra una progresión desde scripts simples hasta sistemas asíncronos con autenticación y persistencia escalable.
