# Contactos API (Flask + MySQL + SQLAlchemy)

Descripción: Aplicación de gestión de contactos que utiliza el ORM SQLAlchemy para interactuar con una base de datos MySQL. Implementa un sistema completo de CRUD con una arquitectura de Blueprints para escalabilidad.

Conceptos Técnicos Aplicados:
> - Uso de Blueprints para modularizar rutas en `Flask`.
> - Mapeo Objeto-Relacional (ORM) con `SQLAlchemy`.
> - Gestión de entornos virtuales y dependencias.
> - Integración con bases de datos relacionales (MySQL).
> - Patrón de diseño de modelos y rutas separadas.

Stack: Python, Flask, MySQL, Flask-SQLAlchemy.

Cómo ejecutar:
1. Instalar dependencias dentro de un entorno virtual.
2. Configurar la cadena de conexión en `config.py`.
3. Ejecutar `python index.py`.

Notas de aprendizaje: Se reforzó el uso de ORMs para abstraer consultas SQL complejas hacia objetos de Python, facilitando la manipulación de datos relacionales.
