# Task Management System (Flask)

Descripción: Sistema integral de gestión de tareas con una arquitectura modular. Utiliza Blueprints para separar las responsabilidades de la aplicación y SQLAlchemy para la persistencia de datos relacionales.

Conceptos Técnicos Aplicados:
> - Organización modular mediante Blueprints.
> - Aplicación del patrón de diseño Model-View-Controller (MVC).
> - Integración de componentes de servidor y recursos estáticos.
> - Gestión de dependencias mediante `requirements.txt`.

Stack: Python, Flask, SQLAlchemy.

Cómo ejecutar:
1. Instalar dependencias: `pip install -r requirements.txt`.
2. Configurar base de datos en `config.py`.
3. Ejecutar `python index.py`.

Notas de aprendizaje: Se reforzó la importancia de una estructura de carpetas limpia y profesional para aplicaciones escalables, facilitando la colaboración y el mantenimiento.
