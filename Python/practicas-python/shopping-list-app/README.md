# Shopping List App (Flask)

Descripción: Aplicación para la gestión de listas de compras organizada por categorías. Utiliza una arquitectura modular para manejar productos y categorías de forma independiente.

Conceptos Técnicos Aplicados:
> - Uso de SQLAlchemy para el modelado de relaciones entre categorías y productos.
> - Implementación de Blueprints para el manejo de rutas modulares.
> - Diseño de interfaces dinámicas con Jinja2.

Stack: Python, Flask, SQLAlchemy.

Cómo ejecutar:
1. Configurar la base de datos en `config.py`.
2. Ejecutar `python app.py`.

Notas de aprendizaje: Se practicó la gestión de relaciones uno-a-muchos en una base de datos relacional y cómo representar estas relaciones en una interfaz de usuario fluida.
