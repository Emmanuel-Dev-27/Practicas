# API Simple (Flask)

Descripción: Una implementación mínima de una API REST utilizando Flask. Sirve como base para entender el enrutamiento y la respuesta en formato JSON sin complicaciones de persistencia externa.

Conceptos Técnicos Aplicados:
> - Definición de rutas y métodos HTTP.
> - Retorno de objetos JSON mediante `jsonify` o diccionarios.
> - Manejo básico de peticiones.

Stack: Python, Flask.

Cómo ejecutar:
1. Ejecutar `python app.py`.

Notas de aprendizaje: Este ejercicio permitió consolidar el concepto de "servidor web" en su forma más pura y sencilla.
