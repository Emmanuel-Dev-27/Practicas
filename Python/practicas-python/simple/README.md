# Ejercicios Simples de Python

Descripción: Colección de scripts introductorios centrados en la lógica algorítmica y las características modernas de Python, como el uso de indicios de tipo (Type Hints).

Conceptos Técnicos Aplicados:
> - Lógica condicional (ej. determinación de años bisiestos).
> - Tipado estático sugerido (`Type Hints`).
> - Entrada y salida estándar de datos.
> - Gestión de entornos virtuales básicos (`venv`).

Stack: Python 3.

Cómo ejecutar:
```bash
python nombre_del_script.py
```

Notas de aprendizaje: Estos scripts sirvieron para practicar la base sintáctica del lenguaje y familiarizarse con las mejores prácticas en el tipado de variables.
