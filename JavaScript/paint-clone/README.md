# MS Paint Clone

Descripción: Una aplicación de dibujo basada en la web que emula las funcionalidades básicas de MS Paint. Permite dibujar a mano alzada, crear rectángulos, borrar y seleccionar colores directamente de la pantalla.

Conceptos Técnicos Aplicados:
> - Uso intensivo de la API de Canvas 2D de HTML5.
> - Implementación del API experimental `EyeDropper` para selección de color.
> - Gestión de estados de dibujo (isDrawing, lastX, lastY).
> - Procesamiento de imágenes en canvas mediante `getImageData` y `putImageData`.
> - Soporte para teclas modificadoras (Shift para proporciones cuadradas).

Stack: HTML, CSS, Vanilla JavaScript.

Cómo ejecutar:
1. Abrir `index.html` en el navegador.

Notas de aprendizaje: Se practicó la manipulación directa de pixeles y trazados vectoriales en el navegador, además de explorar APIs modernas para mejorar la utilidad de herramientas de diseño.
