# TechNews HTML Template

Descripción: Una plantilla de sitio web de noticias tecnológicas responsiva y moderna. Enfocada en el diseño visual limpio y animaciones fluidas de entrada para los elementos de la interfaz.

Conceptos Técnicos Aplicados:
> - Layouts complejos con CSS Grid y Flexbox.
> - Integración de librerías externas mediante CDN (`ScrollReveal`).
> - Diseño responsivo mediante Media Queries.
> - Manipulación de clases para componentes interactivos (menú móvil).

Stack: HTML5, Vanilla CSS, JavaScript, ScrollReveal JS.

Cómo ejecutar:
1. Abrir `index.html` en el navegador.

Notas de aprendizaje: Se reforzó el uso de librerías de animación ligeras para mejorar la "sensación" de la página y la construcción de menús hamburguesa accesibles.
