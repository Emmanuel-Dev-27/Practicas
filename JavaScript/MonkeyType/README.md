# MonkeyType Clone

Descripción: Un clon minimalista de MonkeyType diseñado para practicar velocidad de escritura. Calcula palabras por minuto (WPM) y el porcentaje de precisión en tiempo real, proporcionando una experiencia de usuario fluida y reactiva.

Conceptos Técnicos Aplicados:
> - Manipulación dinámica del DOM mediante plantillas de literales.
> - Manejo avanzado de eventos de teclado (`keydown`, `keyup`).
> - Gestión de temporizadores y lógica de estados de juego.
> - Cálculo estadístico (WPM, Precisión).
> - Uso de módulos de JavaScript para la gestión de datos.

Stack: HTML, CSS, Vanilla JavaScript.

Cómo ejecutar:
1. Abrir `index.html` en cualquier navegador web moderno.
2. Comenzar a escribir las palabras que aparecen en pantalla.

Notas de aprendizaje: Este proyecto permitió profundizar en la gestión de eventos de entrada y en cómo estructurar la lógica de una aplicación basada en el tiempo y la precisión del usuario.
