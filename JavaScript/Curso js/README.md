# Curso de JavaScript - Fundamentos y Objetos

Descripción: Conjunto de ejercicios y lecciones estructuradas para el aprendizaje exhaustivo de JavaScript moderno (ES6+). Cubre desde la lógica de control básica hasta la manipulación avanzada de objetos y patrones de diseño.

Contenido:
- **Control de flujo**: Uso de bucles (`for-of`, `for-in`), estructuras de salto (`break`, `continue`) y selectores (`switch`).
- **Objetos**: Introducción a objetos literales, propiedades dinámicas, funciones de fábrica (Factory Functions) y constructores.
- **Ejercicios**: Resolución de problemas lógicos y algorítmicos para consolidar la sintaxis y el pensamiento computacional.

Stack: Vanilla JavaScript (Node.js o Navegador).

Cómo ejecutar:
Para ejecutar los archivos de consola, use Node.js:
```bash
node nombre_del_archivo.js
```

Notas de aprendizaje: Se reforzó el entendimiento de la naturaleza dinámica de los objetos en JS y la importancia de elegir la estructura de control adecuada para la eficiencia del código.
