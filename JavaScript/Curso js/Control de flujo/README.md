# Módulo: Control de Flujo

Descripción: Profundización en las estructuras que dirigen la ejecución del código en JavaScript.

Conceptos Técnicos Aplicados:
> - Iteración sobre iterables con `for-of`.
> - Enumeración de propiedades de objetos con `for-in`.
> - Control de ciclos con sentencias de interrupción (`break`) y salto (`continue`).
> - Toma de decisiones múltiple con `switch`.

Stack: Vanilla JavaScript.

Notas de aprendizaje: Se practicó la distinción entre iterar valores y enumerar propiedades, clave para evitar errores comunes en la manipulación de colecciones.
