# Módulo: Objetos en JavaScript

Descripción: Estudio de la creación y manipulación de objetos, el pilar central de JavaScript.

Conceptos Técnicos Aplicados:
> - Objetos literales y su naturaleza dinámica.
> - Factory Functions para la creación modular de objetos.
> - Funciones Constructoras y uso de la palabra clave `new`.
> - Atajos de sintaxis (Shorthand properties).

Stack: Vanilla JavaScript.

Notas de aprendizaje: Se comprendió cuándo utilizar constructores versus funciones de fábrica para mejorar la mantenibilidad y organización del código orientado a datos.
