# Prácticas de JavaScript & Frontend

Colección de proyectos web que abarcan desde los fundamentos de Vanilla JS hasta el desarrollo de interfaces modernas con React.

## Proyectos Destacados
- **[React Cursos](./cursos-react)**: Proyectos modulares de aprendizaje en React (Tic-Tac-Toe, Mouse Follower, etc.).
- **[MonkeyType Clone](./MonkeyType)**: Herramienta de medición de velocidad de escritura.
- **[Paint Clone](./paint-clone)**: Aplicación de dibujo con HTML5 Canvas.
- **[JS Core](./Curso%20js)**: Ejercicios profundos de lógica, objetos y control de flujo.

Conceptos Clave: DOM, Hooks (React), Canvas API, Event Loop, Programación Asíncrona.
