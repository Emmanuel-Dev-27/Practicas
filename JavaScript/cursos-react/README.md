# Cursos y Prácticas de React

Descripción: Colección de proyectos enfocados en el aprendizaje progresivo de React, desde la composición de componentes básicos hasta la integración con backends de Python (Flask) y la gestión avanzada de estados y efectos.

Proyectos Incluidos:
- **01-midu**: Conceptos básicos de componentes y props (Twitter Follow Card).
- **02-tic-tac-toe**: Gestión de estados (`useState`), efectos (`useEffect`) y persistencia en LocalStorage.
- **03-mouse-follower**: Ciclo de vida de componentes y limpieza de eventos.
- **04-task-flask**: Integración Full Stack (React + Flask).

Stack: React (Vite), JavaScript (JSX), CSS, Flask (en módulo 04).

Cómo ejecutar:
1. Entrar en la carpeta del proyecto específico.
2. Instalar dependencias: `npm install`.
3. Ejecutar servidor de desarrollo: `npm run dev`.

Notas de aprendizaje: Estas prácticas consolidaron el uso de React como una librería declarativa, enfatizando la importancia de los hooks para la lógica reactiva y la sincronización con APIs externas.
