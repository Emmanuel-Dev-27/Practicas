# Prácticas de Godot Engine - Desarrollo de Videojuegos

Repositorio de prototipos y lecciones de desarrollo de videojuegos utilizando Godot 4 y GDScript.

## Proyectos Destacados
- **[Curso Godot 4](./curso-godot-4-main)**: Guía completa de funcionalidades del motor.
- **[Arkanoid](./arkanoid)**: Clon del clásico con lógica de niveles reactiva.
- **[Pong](./pong)**: Implementación multijugador local.
- **[Prueba de Slider](./pruebaDeSlider)**: Detección cinemática de gestos y sliders.

Conceptos Clave: Signals, Autoloads, KinematicBody, Area2D, Tweens, Animaciones y UI.
