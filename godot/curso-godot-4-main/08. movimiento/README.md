# Módulo: Movimiento y Controladores

Descripción: Implementación de diversas mecánicas de movimiento para personajes, abarcando perspectivas 2D, 2.5D (Pseudo-3D), 3D estándar y visión en primera persona (FPS).

Conceptos Técnicos Aplicados:
> - Uso de `CharacterBody2D` y `CharacterBody3D`.
> - Métodos de movimiento físico (`move_and_slide`).
> - Gestión de gravedad y estados de salto.
> - Captura y procesamiento de `Input` analógico y digital.
> - Implementación de cámaras (SpringArm3D, FPS Camera).

Stack: Godot Engine 4 (GDScript).

Notas de aprendizaje: Se exploró la versatilidad de los nodos de cuerpo físico y cómo adaptar la lógica de movimiento a diferentes géneros y perspectivas de juego.
