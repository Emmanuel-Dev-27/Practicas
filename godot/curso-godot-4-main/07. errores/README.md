# Módulo: Gestión de Errores y Depuración

Descripción: Ejercicios orientados a la identificación, manejo y resolución de errores comunes en el desarrollo con Godot 4.

Conceptos Técnicos Aplicados:
> - Uso de `assert` y `breakpoint`.
> - Impresión de logs y errores personalizados (`push_error`, `push_warning`).
> - Manejo de tipos de datos y prevencion de errores en tiempo de ejecución.

Stack: Godot Engine 4 (GDScript).

Notas de aprendizaje: Se practicaron técnicas de depuración defensiva para asegurar la estabilidad del proyecto y facilitar el mantenimiento del código.
