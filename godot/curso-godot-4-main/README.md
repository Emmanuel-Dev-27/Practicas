# Curso de Godot 4 - Prácticas Completas

Descripción: Este repositorio contiene una serie de proyectos y escenas experimentales diseñadas para dominar las funcionalidades principales de Godot Engine 4. Cubre desde los fundamentos de programación en GDScript hasta sistemas de diseño de niveles avanzados.

Contenido del Curso:
- **04. GDScript**: Fundamentos del lenguaje.
- **05. Autoload**: Gestión de Singletons y estados globales.
- **06. Señales**: Patrón observador y comunicación entre nodos.
- **07. Errores**: Depuración y manejo de excepciones.
- **08. Movimiento**: Controladores de personajes en 2D, 2.5D, 3D y FPS.
- **09. GUI**: Creación de interfaces de usuario y menús.
- **10 & 11. Animación**: Uso de `AnimationPlayer` y grafos de animación.
- **12. Tween**: Animaciones por código y transiciones suaves.
- **13. TileMap**: Diseño de niveles 2D basados en rejillas.
- **14. GridMap**: Diseño de niveles 3D utilizando bibliotecas de mallas.

Stack: Godot Engine 4, GDScript.

Cómo ejecutar: cada carpeta es parte del mismo proyecto principal. Abra el archivo `project.godot` en la raíz de esta carpeta para acceder a todas las escenas.

Notas de aprendizaje: Este conjunto de prácticas representa una progresión completa en el desarrollo de videojuegos, enfocándose en la modularidad y el uso eficiente de las herramientas nativas del motor.
