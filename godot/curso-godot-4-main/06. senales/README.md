# Módulo: Señales (Signals)

Descripción: Práctica centrada en la comunicación desacoplada entre nodos utilizando el sistema de señales nativo de Godot. Permite que los objetos emitan eventos sin conocer quién los escucha.

Conceptos Técnicos Aplicados:
> - Definición de señales personalizadas (`signal`).
> - Conexión de señales vía editor y vía código (`connect`).
> - Paso de argumentos en señales.
> - Patrón de diseño Observador.

Stack: Godot Engine 4 (GDScript).

Notas de aprendizaje: Se practicó cómo estructurar escenas para que sean independientes y reutilizables, evitando referencias directas (`get_node`) en favor de la emisión de eventos.
