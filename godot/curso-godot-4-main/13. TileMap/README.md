# Módulo: TileMap y GridMap

Descripción: Prácticas enfocadas en el diseño de niveles eficientes utilizando sistemas de rejilla tanto en 2D (TileMaps) como en 3D (GridMaps).

Conceptos Técnicos Aplicados:
> - Configuración de `TileSet` y pintado de niveles 2D.
> - Uso de `GridMaps` para diseño modular 3D.
> - Ajuste de colisiones en conjuntos de teselas y mallas.
> - Optimización de dibujo mediante sistemas de instancia masiva.

Stack: Godot Engine 4 (GDScript).

Notas de aprendizaje: Se reforzó la habilidad de construir entornos complejos de forma rápida y escalable, utilizando las herramientas de edición de niveles nativas del motor.
