# Módulo: Animación

Descripción: Estudio de las herramientas de animación de Godot, incluyendo la creación de clips de animación y la integración de grafos de transición para estados de personajes.

Conceptos Técnicos Aplicados:
> - Creación de pistas de animación con `AnimationPlayer`.
> - Mezcla de animaciones y estados mediante `AnimationTree`.
> - Animación de propiedades por keyframes.
> - Control de animaciones mediante código.

Stack: Godot Engine 4 (GDScript).

Notas de aprendizaje: Se practicó la sincronización visual con la lógica del juego, permitiendo que las acciones del jugador se reflejen de forma fluida mediante transiciones animadas.
