# Módulo: Autoload (Singletons)

Descripción: Implementación de nodos que se cargan automáticamente al iniciar el juego, permitiendo la persistencia de datos entre diferentes escenas y el acceso global a variables.

Conceptos Técnicos Aplicados:
> - Configuración de Singletons en los ajustes del proyecto.
> - Gestión de estados globales (ej. puntuación, progreso).
> - Persistencia de datos entre cambios de escena.

Stack: Godot Engine 4 (GDScript).

Notas de aprendizaje: Se comprendió la utilidad de los `Autoloads` para centralizar la lógica de alto nivel y evitar la pérdida de información al cambiar de nivel o escena.
