# Módulo: Fundamentos de GDScript

Descripción: Introducción profunda a la sintaxis y tipos de datos del lenguaje GDScript. Cubre desde variables básicas hasta estructuras de control avanzadas y manipulación de listas/diccionarios.

Conceptos Técnicos Aplicados:
> - Tipado dinámico y estático.
> - Arreglos (`Array`) y Diccionarios (`Dictionary`).
> - Bucles (`for`, `while`) y Estructuras Condicionales (`if`, `elif`, `else`).
> - Definición de funciones y parámetros.
> - Operadores booleanos y comparación.

Stack: Godot Engine 4 (GDScript).

Notas de aprendizaje: Estas prácticas sientan las bases lógicas necesarias para desarrollar cualquier mecánica compleja dentro de Godot, enfatizando el uso eficiente de las estructuras de datos nativas.
