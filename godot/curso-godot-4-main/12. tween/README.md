# Módulo: Tweens

Descripción: Práctica sobre el uso de la clase `Tween` para crear animaciones fluidas mediante código, controlando propiedades de nodos a lo largo del tiempo con diversas curvas de transición.

Conceptos Técnicos Aplicados:
> - Creación y configuración de `Tween`.
> - Métodos de interpolación (`tween_property`).
> - Uso de transiciones (`TransType`) y suavizados (`EaseType`).
> - Secuenciación de animaciones y callbacks.

Stack: Godot Engine 4 (GDScript).

Notas de aprendizaje: Se reforzó la habilidad de crear micro-interacciones y efectos visuales dinámicos sin depender exclusivamente del editor de animaciones.
