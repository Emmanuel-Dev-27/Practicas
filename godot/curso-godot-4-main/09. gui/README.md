# Módulo: Interfaces de Usuario (GUI)

Descripción: Diseño y programación de interfaces gráficas utilizando el sistema de nodos `Control` de Godot. Incluye gestión de menús, barras de estado y layouts responsivos.

Conceptos Técnicos Aplicados:
> - Contenedores (`VBoxContainer`, `HBoxContainer`, `MarginContainer`).
> - Uso de Temas (`Theme`) y recursos de fuentes.
> - Anclajes (Anchors) y márgenes para diseño responsivo.
> - Señales de botones y manipulación de propiedades de UI por código.

Stack: Godot Engine 4 (GDScript).

Notas de aprendizaje: Se aprendió a separar la lógica visual de la lógica del juego y a crear menús que se adapten a diferentes resoluciones de pantalla.
